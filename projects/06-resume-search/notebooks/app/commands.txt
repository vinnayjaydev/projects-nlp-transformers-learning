See running containers
docker ps

Stop the running container
docker stop <container_id>

Run the container in detached (background) mode
docker run -d -p 8000:8000 --name resume-search-container resume-search-api

View logs from the background container
docker logs -f resume-search-container

Remove the container after stopping it
docker rm resume-search-container
